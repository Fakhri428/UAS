# Panduan Cara Kerja Web Human Skill Exchange

Dokumen ini menjelaskan alur penggunaan web dan tombol apa saja yang perlu diklik.

## 1. Halaman awal

Saat web dibuka, kamu akan masuk ke halaman utama.

### Tombol yang ada
- **Log in**: dipakai kalau sudah punya akun.
- **Register**: dipakai untuk membuat akun baru.
- **Dashboard**: muncul kalau sudah login.

### Alur pertama kali
1. Klik **Register**.
2. Isi nama, email, dan password.
3. Setelah akun jadi, login lewat **Log in**.
4. Masuk ke **Dashboard** untuk mulai isi data.

## 2. Dashboard member

Dashboard adalah pusat pengelolaan akun.

### Tombol utama di dashboard
- **Simpan profil**: menyimpan bio, lokasi, mode kerja, waktu tersedia, portfolio, dan social link.
- **Tambah skill**: menambahkan skill baru.
- **Publikasikan offer**: membuat penawaran yang bisa ditukar.
- **Publikasikan need**: membuat kebutuhan bantuan.
- **Simpan portfolio**: menambahkan hasil karya atau project.
- **Cari offer**: pindah ke market untuk melihat penawaran user lain.
- **Lihat match**: melihat rekomendasi partner yang cocok.

### Cara pakai dashboard
#### A. Isi profil dulu
1. Masuk ke dashboard.
2. Isi form **Profil exchange**.
3. Klik **Simpan profil**.

#### B. Tambah skill
1. Isi form **Tambah skill**.
2. Klik **Tambah skill**.

#### C. Buat offer
1. Isi judul offer.
2. Pilih tipe offer.
3. Isi kategori, deskripsi, ekspektasi barter, dan durasi.
4. Klik **Publikasikan offer**.

#### D. Buat need
1. Isi judul need.
2. Isi kategori, deskripsi, dan barter yang ditawarkan.
3. Klik **Publikasikan need**.

#### E. Tambah portfolio
1. Isi judul portfolio.
2. Isi deskripsi hasil karya atau project.
3. Tambahkan link file atau project jika ada.
4. Klik **Simpan portfolio**.

## 3. Halaman market

Halaman market dipakai untuk melihat offer dan need dari member lain.

### Tombol yang ada
- **Cari**: menjalankan pencarian berdasarkan kata kunci dan kategori.
- **Detail**: membuka detail offer.
- **Cari offer**: tombol cepat dari dashboard ke market.
- **Lihat match**: melihat rekomendasi partner.
- **Buat offer**: tombol cepat untuk kembali ke dashboard dan membuat offer.

### Cara pakai market
1. Masukkan kata kunci di kolom pencarian.
2. Pilih kategori jika perlu.
3. Klik **Cari**.
4. Klik **Detail** pada offer yang menarik.

## 4. Halaman detail offer

Di halaman detail offer, kamu bisa membaca isi offer dan mengirim exchange request.

### Tombol yang ada
- **Isi form request**: lompat ke form pengajuan exchange.
- **Login untuk exchange**: jika belum login.
- **Kirim request**: mengirim permintaan exchange.
- **Buat need**: kalau belum punya need sendiri.
- **Cari offer lain**: kalau offer itu milik sendiri.

### Alur kirim request dari offer
1. Buka market.
2. Klik **Detail** pada offer.
3. Jika sudah login, klik **Isi form request**.
4. Pilih **Need saya**.
5. Isi pesan request.
6. Klik **Kirim request**.

## 5. Halaman detail need

Di halaman detail need, kamu bisa menawarkan bantuan ke pemilik need.

### Tombol yang ada
- **Isi form bantuan**: lompat ke form pengajuan exchange.
- **Login untuk exchange**: jika belum login.
- **Kirim request**: mengirim permintaan exchange.
- **Buat offer**: kalau belum punya offer sendiri.
- **Cari need lain**: kalau need itu milik sendiri.

### Alur kirim request dari need
1. Buka market.
2. Klik detail need.
3. Jika sudah login, klik **Isi form bantuan**.
4. Pilih **Offer saya**.
5. Isi pesan request.
6. Klik **Kirim request**.

## 6. Halaman match

Halaman match menampilkan partner yang paling cocok berdasarkan:
- kategori offer dan need,
- kata kunci yang mirip,
- lokasi dan mode kerja,
- reputasi user.

### Tombol yang ada
- **Mulai exchange**: menuju dashboard atau login.

### Cara pakai match
1. Klik **Lihat match** dari dashboard atau buka halaman match.
2. Lihat kartu rekomendasi partner.
3. Klik **Mulai exchange** untuk lanjut ke dashboard dan mulai request.

## 7. Exchange request di dashboard

Semua request yang masuk dan keluar ditampilkan di bagian **Exchange request**.

### Tombol status yang mungkin muncul
- **Accept**: menerima request.
- **Reject**: menolak request.
- **Cancel**: membatalkan request yang masih pending.
- **Mulai progress**: mengubah status ke proses berjalan.
- **Konfirmasi selesai**: menandai bahwa kamu sudah menyelesaikan bagianmu.
- **Menunggu partner**: status informatif, bukan tombol.

### Aturan klik status
- Kalau status masih **pending** dan request masuk ke kamu, klik **Accept** atau **Reject**.
- Kalau status masih **pending** dan request itu kamu kirim, klik **Cancel**.
- Kalau request sudah **accepted**, klik **Mulai progress**.
- Kalau pekerjaan sudah selesai, klik **Konfirmasi selesai**.
- Setelah kedua pihak konfirmasi selesai, status menjadi **completed**.

## 8. Informasi reputasi

Di dashboard ada kartu **Reputasi**.

Isinya:
- jumlah exchange selesai,
- rating rata-rata,
- jumlah review,
- skor trust.

Skor ini dipakai untuk membantu sistem match.

## 9. Fitur plan

Bagian **Limit plan** menunjukkan batas fitur sesuai paket.

### Yang dicek
- jumlah skill,
- jumlah need,
- jumlah offer,
- jumlah request per bulan.

Jika limit habis, sistem akan menolak penambahan data baru.

## 10. Portfolio

Bagian **Portfolio saya** dipakai untuk menyimpan karya, tugas, atau project yang pernah dikerjakan.

### Tombol yang ada
- **Simpan portfolio**: menambahkan portfolio baru.
- **Hapus**: menghapus portfolio yang sudah tidak dipakai.

### Cara pakai
1. Isi form portfolio di dashboard.
2. Klik **Simpan portfolio**.
3. Portfolio baru akan muncul di bagian **Portfolio saya**.
4. Jika ingin menghapus, klik **Hapus**.

## 11. Ringkasan alur paling singkat

1. Klik **Register**.
2. Login lewat **Log in**.
3. Buka **Dashboard**.
4. Klik **Simpan profil**.
5. Klik **Tambah skill**.
6. Klik **Publikasikan offer** atau **Publikasikan need**.
7. Buka **Market**.
8. Klik **Detail** pada offer/need.
9. Klik **Kirim request**.
10. Kelola status request di dashboard dengan **Accept**, **Reject**, **Cancel**, **Mulai progress**, atau **Konfirmasi selesai**.
